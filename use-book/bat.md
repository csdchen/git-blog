## 参数传递

```
%0   - 脚本本身

%~1  - 第一个参数，除去双引号
%*   - 代表所有参数
```

## mklink
- NTFS 符号链接与 Windows 快捷方式文件不同
- mklink [ [/D] | [/H] | [/J] ] Link Target
```
- /D     - 创建目录符号链接
- /H     - 创建硬链接
- /J     - 创建目录连接点
- Link   - 指定新的符号链接名称
- Target - 指定新符号链接引用的路径
```
```
1. 目录连接点只允许 Target 使用绝对路径
2. 目录连接点不属于符号链接
3. 硬连接与软连接
```

##  > nul 2 >&1
```
把标准输出流重定向到nul，把标准错误流重定向到标准输出流。
输出到nul的内容会被直接丢弃，因此效果就是丢弃程序的一切输出
```
