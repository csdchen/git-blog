## 无法登陆  

- 事件: 某角色无法登陆  
```
    时间: 开发阶段  
    问题: 角色面板正常打开，点击登陆时直接掉线  
    分析调试:
        无法连接gateway
        loginserver返回场景不存在错误: gameworld没有向Loginserver注册该场景
    原因: 策划新增场景, 热更逻辑与场景, 在新增的场景下线  
```

- 事件: dataaccess无法正常启动
```
    时间: 开发阶段
    问题：启动时直接崩溃
    分析调试:
        gdb显示崩溃在fastdb申请内存
        config: init_size_mb = 128 -> 134217728
        db-allocate: 134219776
    原因: 配置预设分配的内存大小是128M 实际数据库需要的超过128
```
